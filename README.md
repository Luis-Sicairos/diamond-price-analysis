# Diamond Price Analysis

## Overview

This project analyzes a dataset of diamond characteristics to determine the factors that influence diamond prices. The analysis was completed in **R** as part of a university statistics course and includes exploratory data analysis, correlation analysis, and regression modeling.

## Skills Demonstrated

- R Programming
- Data Analysis
- Exploratory Data Analysis (EDA)
- Data Visualization
- Correlation Analysis
- Simple Linear Regression
- Multiple Linear Regression
- Stepwise Regression (AIC)
- Variance Inflation Factor (VIF)
- Statistical Modeling

## Project Summary

In this project I:

- Selected a random sample of 1,000 diamonds from a larger dataset.
- Explored the data using summary statistics, histograms, and bar charts.
- Examined relationships between variables using a correlation matrix.
- Built simple and multiple linear regression models to predict diamond prices.
- Evaluated model assumptions and interpreted regression results.
- Applied stepwise regression and VIF analysis to improve the model.

## Results

The analysis showed that **carat size** was the strongest predictor of diamond price. The final multiple linear regression model achieved an adjusted **R² of approximately 0.93**, indicating excellent predictive performance.
